Crystal Collector

Here's how you play the game


  1. At the start of the game a random number will appear"Number To Get"
  2. There are four crystals below.
    • Blue
    • Green
    • Red
    • Yellow
  3. Each crystal is worth a different numerical amount
  4. You win the game by matching your total score to the random number.
  5. You lose the game if your total score goes above the random number.
  6. Each time the game starts, all values will change.
